Privacy Policy

Figma Color Extractor

Last updated: July 1, 2026

This extension is designed to extract color information from Figma files and help users export that information as reusable design tokens.

Information We Access

  • Figma file content and related metadata that the user chooses to process in the extension.
  • A Figma personal access token, if the user enters one to enable API access.
  • Local extension settings such as display preferences, recent file history, and cached palette data.

How We Use Information

  • To fetch and analyze Figma file data so the extension can extract colors.
  • To remember user preferences and improve the extension experience.
  • To optionally store a token locally if the user chooses to save it.

What We Do Not Do

  • We do not sell user data.
  • We do not transfer user data to third parties for unrelated purposes.
  • We do not use user data for credit, lending, or advertising profiles.
  • We do not execute remote code.

Data Storage

User settings and optional token storage are kept locally in Chrome extension storage or local browser storage. Cached palette data is also stored locally to improve performance.

Data Sharing

The extension sends data only to the Figma API when the user requests extraction. We do not share personal data with unrelated third parties.

Contact

If you have questions about this privacy policy, contact the publisher using the email listed in the Chrome Web Store.
